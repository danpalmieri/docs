# Subscription

O módulo Subscription fornece endpoints para gerenciar assinaturas na plataforma Clubify.

## Endpoints

### Listar Assinaturas

<ParamField query="limit" type="integer">
  O número máximo de resultados a retornar
</ParamField>

<ResponseField name="data" type="array">
  Lista de assinaturas
</ResponseField>

<ResponseField name="data[].id" type="integer">
  ID da assinatura
</ResponseField>

<ResponseField name="data[].user_id" type="integer">
  ID do usuário
</ResponseField>

<ResponseField name="data[].plan_id" type="integer">
  ID do plano
</ResponseField>

<ResponseField name="data[].status" type="string">
  Status da assinatura (active, canceled, expired)
</ResponseField>

<ResponseField name="data[].start_date" type="string">
  Data de início da assinatura
</ResponseField>

<ResponseField name="data[].end_date" type="string">
  Data de término da assinatura
</ResponseField>

<ResponseField name="data[].created_at" type="string">
  Data de criação da assinatura
</ResponseField>

<ResponseField name="data[].updated_at" type="string">
  Data de atualização da assinatura
</ResponseField>

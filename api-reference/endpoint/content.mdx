# Content

O módulo Content fornece endpoints para gerenciar conteúdo na plataforma Clubify.

## Endpoints

### Listar Conteúdo

<ParamField query="limit" type="integer">
  O número máximo de resultados a retornar
</ParamField>

<ResponseField name="data" type="array">
  Lista de conteúdos
</ResponseField>

<ResponseField name="data[].id" type="integer">
  ID do conteúdo
</ResponseField>

<ResponseField name="data[].title" type="string">
  Título do conteúdo
</ResponseField>

<ResponseField name="data[].description" type="string">
  Descrição do conteúdo
</ResponseField>

<ResponseField name="data[].created_at" type="string">
  Data de criação do conteúdo
</ResponseField>

<ResponseField name="data[].updated_at" type="string">
  Data de atualização do conteúdo
</ResponseField>

# Invoice

O módulo Invoice fornece endpoints para gerenciar faturas na plataforma Clubify.

## Endpoints

### Listar Faturas

<ParamField query="limit" type="integer">
  O número máximo de resultados a retornar
</ParamField>

<ResponseField name="data" type="array">
  Lista de faturas
</ResponseField>

<ResponseField name="data[].id" type="integer">
  ID da fatura
</ResponseField>

<ResponseField name="data[].user_id" type="integer">
  ID do usuário
</ResponseField>

<ResponseField name="data[].payment_id" type="integer">
  ID do pagamento
</ResponseField>

<ResponseField name="data[].invoice_number" type="string">
  Número da fatura
</ResponseField>

<ResponseField name="data[].amount" type="number">
  Valor da fatura
</ResponseField>

<ResponseField name="data[].currency" type="string">
  Moeda da fatura
</ResponseField>

<ResponseField name="data[].status" type="string">
  Status da fatura (draft, issued, paid, canceled)
</ResponseField>

<ResponseField name="data[].issue_date" type="string">
  Data de emissão da fatura
</ResponseField>

<ResponseField name="data[].due_date" type="string">
  Data de vencimento da fatura
</ResponseField>

<ResponseField name="data[].created_at" type="string">
  Data de criação da fatura
</ResponseField>

<ResponseField name="data[].updated_at" type="string">
  Data de atualização da fatura
</ResponseField>

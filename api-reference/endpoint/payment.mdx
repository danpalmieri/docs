# Payment

O módulo Payment fornece endpoints para gerenciar pagamentos na plataforma Clubify.

## Endpoints

### Listar Pagamentos

<ParamField query="limit" type="integer">
  O número máximo de resultados a retornar
</ParamField>

<ResponseField name="data" type="array">
  Lista de pagamentos
</ResponseField>

<ResponseField name="data[].id" type="integer">
  ID do pagamento
</ResponseField>

<ResponseField name="data[].user_id" type="integer">
  ID do usuário
</ResponseField>

<ResponseField name="data[].subscription_id" type="integer">
  ID da assinatura
</ResponseField>

<ResponseField name="data[].amount" type="number">
  Valor do pagamento
</ResponseField>

<ResponseField name="data[].currency" type="string">
  Moeda do pagamento
</ResponseField>

<ResponseField name="data[].status" type="string">
  Status do pagamento (pending, completed, failed, refunded)
</ResponseField>

<ResponseField name="data[].payment_method" type="string">
  Método de pagamento
</ResponseField>

<ResponseField name="data[].payment_date" type="string">
  Data do pagamento
</ResponseField>

<ResponseField name="data[].created_at" type="string">
  Data de criação do pagamento
</ResponseField>

<ResponseField name="data[].updated_at" type="string">
  Data de atualização do pagamento
</ResponseField>
